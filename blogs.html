Difference between Map, forEach, Filter And Find

  • map() will return a new array as per the conditions applied.
  • forEach() will not return anything. forEach() returns undefined.
  • filter() method will return an array of matching elements else will return an empty array if no matching happens.
  • find() will return a single first matching elements according to the condition.

Difference between Local Storage, Session Storage And Cookie

  • local Storage stores up to 10 megabytes. Local storage expire never.
  • session storage stores up to 5 megabytes. Session storage expire on a tab close.
  • Cookies can store very less 4 kilobytes of data. Cookies are manually set.

What is event loop and how it is works.

event loop works in a cyclic manner, where it continually checks whether or not the call stack is empty. If it is empty, new functions are added from the event queue. If it is not, then the current function call is processed.
